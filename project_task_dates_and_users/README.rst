============================
Project task dates and users
============================

* In project task 4 new fields: "Date done", "User date done", "Revised date",
  and "User revised date".
* The person who puts the "Date done" should appear next to it automatically,
  and the same with "Date Revised".

Bug Tracker
===========

Bugs are tracked on `GitHub Issues
<https://github.com/avanzosc/project-addons/issues>`_. In case of trouble,
please check there if your issue has already been reported. If you spotted
it first, help us smash it by providing detailed and welcomed feedback.

Do not contact contributors directly about support or help with technical issues.
